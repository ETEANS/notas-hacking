
**Reto:**
-Cambiar la extensión de un archivo para obtener la bandera.

**Descripción:**
-This is a really weird text file [TXT](https://jupiter.challenges.picoctf.org/static/e7e5d188621ee705ceeb0452525412ef/flag.txt)? Can you find the flag?

**Solución:**
Cambiar la extensión del archivo de texto a .png para abrir la imagen y obtener la bandera.
-Descargar el archivo.
-Cambiar la extensión del archivo a .png.
-Abrir la imagen y copiar la bandera.

El resultado es: **picoCTF{now_you_know_about_extensions}**.

**Notas adicionales:**
-Las imagenes de tipo PNG tienen en las primeros bytes el texto 'PNG'.
-Es necesario activar la opción de cambiar las extensiones.
